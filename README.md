# README

## Notes

Go to SonarQube UI: http://localhost:9000/projects
Default user: admin / admin 

## Getting Started

**Start SonarQube**
```sh
docker-compose up -d
```
### Install Java Plugin
1. Go to Administration -> MarketPlace
2. Look for "SonarJava" and Install


**Stop SonarQube**
```
docker-compose down
```
